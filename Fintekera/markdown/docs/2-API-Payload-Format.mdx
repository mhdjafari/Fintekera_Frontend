## Input Parameters

#### Payload
Body Example:

```sh

inputData =
    {"gross_income": 200000,
    "zip_code":"89061",
    "job_title": "Engineer",
    "monthly_debt": 150,
    "requested_monthly_payment": 300,
    "experience_year": 2,
    "company_name": "Google",
    "industry_name": "Internet/Online"
    }

```

#### Features' Detail:

**gross_income:**
 - **Description:** An integer variable representing the total annual taxable income of an individual.
 - **Valid Range:** 0 to 99,999,999

**zip_code:**
- **Description:** A string variable representing the postal code, consisting of 5 numeric characters.
- **Valid Range:** 00000 to 99999

**job_title:**
- **Description:** A string variable indicating the job title of an individual.
- **Valid Range:** N/A

**monthly_debt:**
- **Description:** An integer variable specifying the monthly recurring debt amount.
- **Valid Range:** 0 to 99,999,999

**experience_year:**
- **Description:** An integer variable representing the number of years of experience in the current position.
- **Valid Range:** 0 to 12

**requested_monthly_payment:**
- **Description:** An integer variable specifying the expected monthly payment to assess the ability to pay.
- **Valid Range:** 0 to 99,999,999

**company_name:**
- **Description:** A string variable indicating the name of the employer.
- **Valid Range:** N/A

**industry_name:**
- **Description:** A string variable specifying the industry of the employer.
- **Valid Range:** N/A
